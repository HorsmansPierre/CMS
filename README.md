# KuLaDig CMS Website – Abgabe

Dieses Repo enthält die Abgabe zum CMS-Projekt: Dokumentation (PDF), Konzept (PPT/PDF) sowie den WordPress-Stand als ZIP (Themes und verwendete Plugins).  
Installation: ZIP entpacken und themes und plugins nach `wp-content/` kopieren, anschließend Theme im Backend aktivieren.  
Die Inhalte werden lesend über die öffentliche KuLaDig-API geladen; Performance wird über Caching (Transients) verbessert.
